JKYAdvertiserErrorCode

All errors returned by methods of this SDK are in the JKYAdvertiserErrorDomain error domain and have one of the following error codes.

In most cases, the errors contain a human-readable description in NSError’s localizedDescription (although that description is only provided in English - thus, you should not display those error descriptions to your users - they are rather intended to give developers using this SDK a better chance to understand what’s wrong).
